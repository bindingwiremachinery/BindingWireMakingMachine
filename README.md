# Premium Industrial Machinery Platform

Technical website deployment for Indian Nail Machine, based in Rajkot, Gujarat.

## 🚀 Deployment Instructions
1. Upload `index.html`, `manifest.json`, `robots.txt`, `sitemap.xml`, and `.htaccess` directly into the public root directory.
2. Ensure DNS records point correctly via the `CNAME` file parameters.

## ⚙️ Target SEO Core Features
* Automated JSON-LD dynamic Product schema integration for IN-series machinery blocks.
* Multi-stage caching headers activated via root server configurations.
